Project Overview

The Emotion Detection System utilizes convolutional neural networks (CNNs) to classify facial expressions into various emotional categories, such as happy, sad, angry, and more.

Details

  • Problem Statement: Automatically detect and classify human emotions based on facial expressions.
  • Key Components: Preprocessing images, data augmentation, CNN-based classification, and performance evaluation.
  • Libraries Used: NumPy, OpenCV, Keras, matplotlib, scikit-learn
  • Skills Gained: Deep learning with CNNs, emotion classification, and confusion matrix analysis.

Key Features

  • Preprocesses grayscale images from the FER-2013 dataset, normalizing pixel values and resizing images.
  • Uses data augmentation techniques to improve model generalization.
  • Trains a convolutional neural network with dropout for regularization and uses softmax for multi-class classification.
  • Evaluates model performance using accuracy metrics and visualizes the training process with accuracy plots.
  • Generates and visualizes a confusion matrix to understand model performance across emotion categories.
