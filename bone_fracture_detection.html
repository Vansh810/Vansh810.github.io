Bone Fracture Detection

Back to Projects

Project Overview

This project employs transfer learning to detect bone fractures in medical images. The pre-trained Xception model was fine-tuned to classify images as fractured or non-fractured, providing accurate predictions for medical diagnostics.

Details

  • Problem Statement: Detect bone fractures in medical images using a transfer learning-based approach.
  • Problem Type: Binary Classification
  • Libraries Used: TensorFlow, Keras, NumPy, Matplotlib
  • Skills Gained: Transfer learning, model fine-tuning, medical image processing, performance metrics evaluation, and data augmentation

Key Features

  • Utilized the Xception model pre-trained on ImageNet as the base for transfer learning.
  • Fine-tuned specific layers of the base model for domain-specific knowledge adaptation.
  • Applied data augmentation techniques to improve model generalization and performance.
  • Implemented early stopping to prevent overfitting during training.
  • Evaluated the model using metrics such as accuracy, precision, recall, and AUC-ROC.
  • Visualized training progress and validation metrics over epochs for model optimization.
