Project Overview

This project focuses on predicting the age of a person based on their facial image using deep learning techniques. A convolutional neural network (CNN) was implemented for this regression problem, estimating the age as a continuous value.

Details

  • Problem Statement: Develop a deep learning model to predict the age of individuals from facial images.
  • Problem Type: Regression
  • Libraries Used: NumPy, OpenCV, TensorFlow, Keras, Matplotlib, scikit-learn
  • Skills Gained: Deep learning for regression, image preprocessing, data augmentation, performance evaluation, and model optimization

Key Features

  • Designed and implemented a CNN model for age regression using TensorFlow/Keras.
  • Preprocessed the dataset by extracting age labels from file names and resizing images to a uniform format.
  • Normalized pixel values and augmented the data to enhance model generalization.
  • Trained the model with mean absolute error (MAE) as the evaluation metric for accurate predictions.
  • Visualized model performance through training and validation MAE plots.
  • Tested the model on unseen images for real-world performance evaluation.
