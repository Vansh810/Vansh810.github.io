Project Overview

This project allows users to query documents using advanced language models. By loading and processing documents, the system provides precise and contextual answers to user queries.

Details

  • Problem Statement: Enable users to ask questions about a document and receive contextually accurate answers.
  • Key Components: Document processing, text chunking, vector embeddings, and language model-based retrieval.
  • Libraries Used: LangChain, HuggingFace, tkinter, dotenv
  • Skills Gained: Natural Language Processing (NLP), embedding creation, vector search, and interaction with modern LLMs.

Key Features

  • Supports text and PDF documents through manual file selection.
  • Processes large documents by splitting them into manageable chunks for efficient query retrieval.
  • Uses HuggingFace embeddings for semantic search and Google Flan-T5 as the language model for Q&A.
  • Interactive query interface for users to ask and refine their questions.
  • Securely loads HuggingFace API credentials from environment variables.
