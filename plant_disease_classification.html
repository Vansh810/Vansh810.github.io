Project Overview

This project focuses on detecting plant diseases using images and deep learning techniques. It leverages convolutional neural networks (CNNs) for image classification and helps in early disease detection to reduce agricultural losses.

Details

  • Problem Statement: Develop a deep learning model to classify plant diseases using image data.
  • Problem Type: Classification
  • Libraries Used: NumPy, OpenCV, TensorFlow, Keras, Matplotlib, scikit-learn
  • Skills Gained: Deep learning model development, image preprocessing, data augmentation, performance evaluation

Key Features

  • Built a CNN model using TensorFlow/Keras for plant disease classification.
  • Preprocessed plant images by resizing, normalizing, and labeling from a custom dataset.
  • Implemented data augmentation to improve model generalization.
  • Trained the model on 10 plant disease categories, achieving high classification accuracy.
  • Evaluated the model using confusion matrix and validation accuracy metrics.
  • Designed a GUI for selecting plant images and predicting the disease type with confidence scores.
