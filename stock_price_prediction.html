Project Overview

The Stock Price Prediction System utilizes historical stock market data to forecast future prices using Prophet, a time-series forecasting model. The system detects growth trends and includes seasonality components for accurate predictions.

Details

  • Problem Statement: Forecast stock prices for better decision-making in investments and trading.
  • Key Components: Data retrieval, growth detection, time-series forecasting, and visualization.
  • Libraries Used: yfinance, Prophet, seaborn, matplotlib, pandas, NumPy.
  • Skills Gained: Time-series analysis, seasonality modeling, stock data handling, and trend detection.

Key Features

  • Retrieves historical stock data using Yahoo Finance API for flexible symbol and date range inputs.
  • Automatically detects growth trends in stock prices and adjusts model parameters accordingly.
  • Integrates seasonality components such as yearly, weekly, and daily patterns to enhance prediction accuracy.
  • Forecasts stock prices up to one year into the future and visualizes trends alongside historical data.
  • Identifies key peaks (maximum and minimum predicted prices) and annotates them on the forecast plot for better insights.
